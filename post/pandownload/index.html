    自建服务器的方法
    在大佬的基础上修改(更准确的教程)（视频教程刚挂不久）
    链接：https://www.bilibili.com/read/cv5641021

hosts文件加上
127.0.0.1 pandownload

下载好文件之后，在文件目录(api同目录下）
新建一个start.bat和start-server.ps1

start.bat
start "" "你的start-server.ps1目录\start-server.ps1"
TIMEOUT /T 1
start "" "你的网盘目录\PanDownload.exe"

start-server.ps1
python -m http.server --bind 0.0.0.0 80

这些就是启动文件了，具体教程见上面的那个链接。
